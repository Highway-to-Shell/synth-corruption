# Phase 1 Summary

Phase 1 created the first playable version of Synth Corruption.

## Completed

- Initial Canvas project shell
- Local TTF text rendering through `js/text.js`
- Game state machine
- Keyboard and mouse input
- Player movement and shooting
- Basic warptime
- Chaser enemies
- Collision and game over flow
- Score and best score
- Lightweight hit and death particles
- Phase 1 test checklist

## Development Order

1. Scaffold the Canvas shell.
2. Add UI, font handling, and the state machine.
3. Add input, player movement, and bullets.
4. Add enemies and gameplay tests.
5. Integrate the phase 1 playable loop.

## Next Phase

- Add corruption cores.
- Add corruption cells that slow the player.
- Add wave triggers.
- Add stronger enemy types.
- Add audio feedback.
- Add CRT and glitch post-processing.
